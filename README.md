# MagTec Invest Website

A modern, responsive website for MagTec Invest showcasing their technology, media, and investment services.

## Files
- `index.html` - Main HTML file
- `style.css` - All CSS styles
- `script.js` - JavaScript functionality

## Features
- Fully responsive design
- Accessible (ARIA labels, keyboard navigation)
- Form validation and submission
- Interactive project modals
- Smooth scrolling
- Performance optimized

## Deployment
Simply upload all three files to any web hosting service. The website requires no build process or dependencies.

## Browser Support
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
